Olá! Meu nome é Anderson, sou estudante de Ciência da Computação e estou praticando as minhas habilidades com HTML, CSS e Javascript.

O projeto a seguir é um jogo de adivinhar número secreto desenvolvido junto com o curso da Alura, com o objetivo de praticar lógica de programação, manipulação do DOM e interatividade utilizando Javascript.

No início do jogo, um número secreto é escolhido aleatoriamente e o usuário precisa tentar adivinhar qual é esse número. A cada tentativa errada, o sistema fornece dicas informando se o número secreto é maior ou menor, até que o jogador consiga acertar.

Além disso, também é possível reiniciar o jogo e jogar novamente quantas vezes quiser.

ENGLISH 

Hello! My name is Anderson, I am a Computer Science student, and I am practicing my skills with HTML, CSS, and Javascript.

The following project is a secret number guessing game developed alongside the Alura course, aiming to practice programming logic, DOM manipulation, and interactivity using Javascript.

At the beginning of the game, a secret number is randomly selected and the user must try to guess what the number is. With each wrong attempt, the system provides hints telling whether the secret number is higher or lower until the player guesses correctly.

In addition, it is also possible to restart the game and play again as many times as desired.
